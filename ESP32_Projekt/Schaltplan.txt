SCHRITT-FÜR-SCHRITT VERKABELUNG:
══════════════════════════════════

1. ESP32 auf Breadboard stecken (oder daneben legen)

2. GND Rail vorbereiten:
   - Eine Reihe auf dem Breadboard als GND markieren (-)
   - ESP32 GND mit dieser Rail verbinden (schwarzes Kabel)

3. Button 1 (GPIO 12):
   - Ein Bein des Buttons → GPIO 12 (rotes Kabel)
   - Anderes Bein → GND Rail (schwarzes Kabel)

4. Button 2 (GPIO 13):
   - Ein Bein → GPIO 13 (oranges Kabel)
   - Anderes Bein → GND Rail

5. Button 3 (GPIO 14):
   - Ein Bein → GPIO 14 (gelbes Kabel)
   - Anderes Bein → GND Rail

6. Button 4 (GPIO 27):
   - Ein Bein → GPIO 27 (grünes Kabel)
   - Anderes Bein → GND Rail

7. Button 5 (GPIO 32):
   - Ein Bein → GPIO 32 (blaues Kabel)
   - Anderes Bein → GND Rail

8. Button 6 (GPIO 33):
   - Ein Bein → GPIO 33 (violettes Kabel)
   - Anderes Bein → GND Rail


ELEKTRISCHES SCHEMA:
═══════════════════════

Jeder Button funktioniert so:

    3.3V (intern, Pull-up)
         │
         ▲ 10kΩ (intern im ESP32)
         │
         ├──── GPIO Pin (wird vom ESP32 gelesen)
         │
        ┌┴┐
        │ │  ← Button (offen = HIGH, geschlossen = LOW)
        └┬┘
         │
        GND


WICHTIGE HINWEISE:
═══════════════════

✓ KEINE externen Widerstände nötig
✓ Button-Orientierung egal (nicht polarisiert)
✓ Reihenfolge der Buttons egal
✓ Kabelfarben nur Vorschlag (nutze was du hast)
✓ GND Rail kann - oder + Reihe sein (Beschriftung egal)

✗ NICHT 5V verwenden! ESP32 ist 3.3V!
✗ NICHT Pin direkt auf 3.3V ziehen


MATERIAL-LISTE:
═══════════════

□ 1x ESP32 Dev Board
□ 1x Breadboard (mindestens 400 Löcher)
□ 6x Taster/Push-Buttons (4-Pin oder 2-Pin)
□ 7x Jumperkabel Male-to-Male (oder mehr)
□ 1x Micro-USB Kabel
□ Optional: Beschriftungsetiketten für Buttons


TEST-SEQUENZ:
═════════════

1. Nur Button 1 verkabeln
2. Code hochladen
3. Serial Monitor öffnen
4. Button 1 drücken → sollte [T12-10] zeigen
5. Funktioniert? → Nächsten Button verkabeln
6. Wiederholen bis alle 6 Buttons funktionieren


ERWEITERTE OPTIONEN:
═══════════════════════

• LED pro Button:
  - LED-Anode → GPIO Pin (mit 220Ω Widerstand)
  - LED-Kathode → GND
  - Im Code: digitalWrite(pin, HIGH) beim Drücken

• Button mit LED-Beleuchtung:
  - Viele Arcade-Buttons haben integrierte LEDs
  - Separate Ansteuerung möglich

• Längere Kabel:
  - Bis ca. 30cm problemlos
  - Bei mehr: 100nF Kondensator am Button gegen Störungen
